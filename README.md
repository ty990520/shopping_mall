## 04. 구현된 기능
1. 회원가입
<br>-아이디중복x
2. 로그인
<br>-로그인성공
3. 마이페이지
<br>-정보수정
4. 포인트(바로충전)
<br>-포인트충전완료
5. 제품리스트
<br>-카테고리
<br>-검색기능
<br>-정렬 ->> 좋아요순
6. 디테일
<br>-제품정보확인
<br>-좋아요기능(중복x, 전체누적좋아요수)
7. 좋아요기능
<br>-마이페이지에서 좋아요목록 확인
8. 개별구매
<br>-포인트충전 후 구매
<br>-바로구매
9. 장바구니구매
  <br>-중복제품 안들어감
  <br>-장바구니 한 네개정도 넣고
  <br>-장바구니 개별삭제
  <br>-장바구니 제품 선택 금액합계
  <br>-포인트충전 후 구매
  <br>-바로 구매
  <br>-장바구니 세션 삭제
  <br>-마이페이지 구매목록
10. 고객센터
<br>-페이지네이션(이전페이지, 다음페이지 기능)
11. 어드민
<br>-제품등록
<br>-제품수정(첨부파일 기존 값 유지)
<br>-제품삭제
<br>-게시글등록

<br>
